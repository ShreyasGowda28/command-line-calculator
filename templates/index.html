Available Features:

  • Basic Operations: +, -, *, /, %, ** (e.g., 2 + 3)
  • Square Root: Use sqrt() (e.g., sqrt(16))
  • Trigonometry: sin, cos, tan (e.g., sin(pi/2))
  • Logarithms: log() for natural logarithm (e.g., log(e))
  • Exponentials: exp() for exponential (e.g., exp(2))
  • Factorial: Use ! for factorial (e.g., 5!)
  • Constants: pi (3.14159) and e (2.71828)
  • Algebraic Expressions: Use variables like x, y, z (e.g., x + 2, x^2 - 5x + 6)
